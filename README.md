다운로더 내부의 yt-dlp를 교체하는 플러그인(최신 버전 + 유튜브 JS challenge 자동 적용)

1. 플러그인(스크립트 적용)
[youtube_enable_ejs.hds](https://github.com/depth221/yt-dlp-default-ejs-github/blob/main/youtube_enable_ejs.hds)에서 스크립트를 다운로드한 후 다운로더에서 옵션 → 설정 → 플러그인으로 들어가 다운로드한 파일을 적용

2. Deno 설치
(윈도우의 경우) 검색창에서 '터미널' 검색 후 우클릭  → 관리자 권한으로 실행

`winget install DenoLand.Deno` 입력 후 엔터

-----
This is a plugin that replaces the built-in yt-dlp in the downloader (latest version + automatic application of the YouTube JS challenge)

1. Plugin (apply script)
Download the script from [youtube_enable_ejs.hds](https://github.com/depth221/yt-dlp-default-ejs-github/blob/main/youtube_enable_ejs.hds), then go to Options → Settings → Plugins in the downloader and apply the downloaded file

2. Install Deno: (For Windows) Search for ‘Terminal’ in the search bar, right-click → Run as administrator.

Enter `winget install DenoLand.Deno` and press Enter.
